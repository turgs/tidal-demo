TIDAL Player Web Components Example

This is a small example to test out the provided web components.

Credentials

Before any of the below works, submit your clientId and clientSecret for your TIDAL Third-Party client.

<tidal-video-view>

Description

Outputs video data when a video is played.

Options

attribute value comment
video-id productId Set this to display a post image when the video is not playing.

Example

<tidal-progress-bar>

Description

Shows the elapsed progress of the playing item. 60 FPS smooth like butter.

Options

No specific attribute options. Fills the box its given. Color controlled by currentColor.

Example

<tidal-current-time>

Description

Shows the current progress time in (hh:)mm:ss format.

Options

No specific attribute options. Fills the box its given. Color controlled by currentColor.

Example

<tidal-duration-time>

Description

Shows the duration of the current media product in (hh:)mm:ss format.

Options

No specific attribute options. Fills the box its given. Color controlled by currentColor.

<tidal-play-trigger>

Description

Triggers playback of a new item and acts as a play/pause button.

Options

attribute value comment
product-id productId Product ID to be played
product-type 'track' | 'video' Product type to be played
